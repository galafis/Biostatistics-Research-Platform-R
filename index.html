Biostatistics Research Platform R

Bem-vindo à Plataforma de Pesquisa em Bioestatística em R. Este projeto oferece um conjunto robusto de ferramentas para análise estatística avançada, visualização de dados e relatórios.

Visão Geral

Esta plataforma é projetada para facilitar a pesquisa em bioestatística, fornecendo funcionalidades para:

  • Carregamento e processamento de dados
  • Estatísticas descritivas
  • Análise de correlação
  • Testes de hipóteses (T-test, ANOVA)
  • Regressão linear
  • Visualização de dados com ggplot2

Para mais detalhes sobre o projeto, incluindo instruções de instalação e uso, por favor, visite o repositório GitHub.
